1.Balloon Tooltip Sample
  A sample code demonstrates how to use Balloon ActiveX control to display a balloon tooltip for
  any control.

2.Get All system Tray Icons Information Sample
  A sample code demonstrate how to use Traylist ActiveX control to get all system tray icons
  Information as well as how to hide a specified icon from system tray area.

3.SubClass Sample
  A sample code demonstrates how to use Subclass ActiveX control to subclass any Form.

4.TrayIcon Sample
    A sample code demonstrate how to add your software icon to system tray area
    As well as how to call the advanced features related to system tray area from within
    your application.
