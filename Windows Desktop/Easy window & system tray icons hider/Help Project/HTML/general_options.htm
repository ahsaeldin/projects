General Options:

 

•Starts when Windows starts: Enable you to Start Easy Window & System Tray Icons Hider at the windows startup of the current user.

 

•Change hidden window name's color to gray in the list: Check this box if you want to change the hidden window caption in the list to gray.

 

•Reverse Window state by click: Change the window state from hide to show or from show to hide by clicking its name in the list.

 

•Automatically reverse window state when the cursor remains over the window name for a certain period of time: Just reverse window state when you hover the window caption with the cursor.

 

•Hide me when hide all windows: Hides Easy Window & System Tray Icons Hider if you Hide all windows using Hide All button or using the shortcut key assigned to Hide All Windows.

 

•Show me as a top most window: Always display Easy Window & System Tray Icons Hider as a top most window in the desktop.

 

•Show Easy Window & System Tray Icons Hider in system tray:

 

  Always: Always display Easy Window & System Tray Icons Hider icon in the system tray.

 

  When minimized: Only display Easy Window & System Tray Icons Hider if minimized.

 

  Never: Never display Easy Window & System Tray Icons Hider in the system tray.

 

•Minimize to system tray instead of closing: Check this box if you want to minimize Easy Window & System Tray Icons Hider to system tray instead of closing it.
