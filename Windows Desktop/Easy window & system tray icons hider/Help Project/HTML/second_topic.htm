Features

•Hide any window either from Easy Window & System Tray Icons Hider itself or using a shortcut key.
•Hide all displayed windows either from Easy Window & System Tray Icons Hider or using a shortcut key.
•Hide any or all icons in system tray area.
•The ability to customize shortcut keys.
•Hide the Foreground window using a shortcut key.
•Revert the State of windows or system tray icons between show and hide.
•Easy Window & System Tray Icons Hider remembers the hidden windows if you close and re-open it.
•Easy Window & System Tray Icons Hider remembers the hidden system tray icon if you close and re-open it.
