Entering Registration Key

 Top  Previous  Next

 

When Handy Startup Monitor is purchased you will be sent an email containing your unique registration key. This registration key must be entered into Handy Startup Monitor to remove the 15-day trial restriction. A licence must be purchased for each user (each person that uses the software).

 

HOW-TO Enter your registration key:

 

Once you have received your registration key perform the following steps:

1.Start Handy Startup Monitor.

2.If you are presented with the evaluation window click on the Enter Key button.

3.If you are not presented with the evaluation window or Handy Startup Monitor is already running click Help button  and select "Enter Registration Key" from the pull-down menu.

4.Enter your registration details exactly as provided. We recommend copying and pasting the name and key from your registration email instead of manually typing it to avoid any errors.

5.Click the OK button.
