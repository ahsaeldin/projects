Record Macro

 Top  Previous  Next

 

To start recording click the "Record" command button or Right Click Perfect Macro Recorder System Tray Icon and Choose Record from the pop up menu.

 

 

When you have finish recording your activity, press Alt+F10. OR press Alt+Ctrl+Del

 

After you have finished recording, Perfect Macro Recorder will ask you to choose a name for the macro, small

description and a global hot key to playback this macro.

 

 

 

Note:

•Press Pause Button if you want to pause recording.
•Press Alt+Insert to Resume recording.
