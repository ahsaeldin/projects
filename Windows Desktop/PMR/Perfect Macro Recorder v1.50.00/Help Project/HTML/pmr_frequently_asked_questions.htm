What is the macro?
A sequence of keyboard strokes and mouse actions that can be recorded so that their playback can be activated by a single keystroke, keystroke combination, or mouse click.

What are the system requirements?
Intel Pentium, Celeron, AMD or compatible processor Windows 98/ME/NT4/2000/XP/2003/Vista

How can I stop recording?
Press Alt+F10.

Is there is a version of Perfect Macro Recorder for Mac or Linux?
No, currently Perfect Macro Recorder can only run under the windows.

How can I stop play backing?
Press Alt+F10 or Ctrl+Alt+Del or Ctrl+Esc.

I'd like to use Perfect Macro Recorder under windows 95, is it possible?
No, Perfect Macro Recorder only run under Windows 98 or above.

Why does Perfect Macro Recorder stop recording/play backing a macro when I press Ctrl+Esc or Ctrl+Alt+Delete?
Because of Windows limitations.

Why Sometimes the high playback speed causes some unexpected results?
Because  in the high play back speed, there is no delay time between events.

What is the executable macro?
You can convert any macro to an EXE file, so the macro can be run stand a lone without Perfect Macro Recorder.

How can I terminate play backing an Executable macro file?
Press Ctrl+Esc keys or Alt+Ctrl+Del keys.
