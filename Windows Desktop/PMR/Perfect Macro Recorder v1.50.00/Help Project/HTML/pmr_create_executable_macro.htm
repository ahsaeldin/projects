Executable Macros

 Top  Previous  Next

With this feature you can convert any macro to an executable file ".exe" that can be run stand a lone without Perfect Macro Recorder

To convert a macro to an executable macro:

1. Choose macro name from the combo box 

2. Click Export to EXE command button to create an Executable macro.

Note that you can't use the keyboard or the mouse until the playback finish.

If you wish to terminate play backing an Executable macro you can choose any of the following methods:

•Press "Ctrl+Esc"
•Press "Alt-Ctrl-Del"
