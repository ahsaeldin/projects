General Options

Play Back Speed:

Choose the play back speed [High or normal or low speed].

 

Record Mouse:

If this option is selected, Perfect Macro Recorder will record mouse events whenever you start recording a new macro.

 

Record keyboard events:

If this option is selected, Perfect Macro Recorder will record keyboard events whenever you start recording a new macro.

 

Show Tips:

    Selecting this options, will display a small tip window when you start recording or start play back.

 

Auto run on windows startup:

       If this option is selected, Perfect Macro Recorder will run at windows startup.

 

Minimize to system tray on Startup:

If this option is selected, Perfect Macro Recorder will minimize to system tray on startup.

 

Repeat Play back:

 

You can choose to play back macros in a one of the following ways:

1.Repeat play back a certain number of time.

2.Repeat play back for a specified period (for example: running a macro for two hours).

3.Repeat play back forever until you stop it using stop playback shortcut key.
