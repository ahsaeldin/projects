When you click My Macros Button the following window will be displayed where you can rename or delete or delete all macros you had recorded before.

•Macro Name Column: The name of your macro.
•Date Column: The Date you create the macro.
•Description Column: The Description you added when you saved the macro.
•Time Column: The Time you create the macro.
•Shortcut Key Column: indicate the shortcut key of the macro if you specified one. you can rename the macro or delete it or delete all the macros

If you want to specify a shortcut key for the macro click the shortcut key button or if you want to disable the shortcut key for a macro.
