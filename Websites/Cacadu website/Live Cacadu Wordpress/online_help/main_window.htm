Main Window

Cacadu's Main Application Window gives the user full view and access to the ribbons, tool bars, and menus the application offers.

From this main window the user will see the following:-

 

 

1.My Tasks Menu on the left

 

2.Task Properties on the right

 

3.Tasks List

 

4.Task Flow of the selected task

 

5.The Ribbon Toolbar at the top

 

6.Status Bar on the bottom

 

7.History Panel

 

8.Error Panel.

 

9.Main Menu

 

10.Quick Access Toolbar

 

See Cacadu's documentation for detailed descriptions of each of the menus, ribbons, tabs, and toolbars accessible from the main menu.
