Options

Cacadu provides the following configuration settings in the option window to control the application behavior as desired...

 

•Cacadu Enabled

 

  It controls the triggers in Cacadu. When this option is unchecked, none of the triggers will execute and restrict the manual execution of tasks.

 

•Starts at Windows StartUp

 

   The user can control if Cacadu will start with windows startup using this option.

 

•Disable Splash Screen

 

  This option controls the splash screen of  Cacadu.

 

•Send errors report automatically in the future

 

  Cacadu has the capability to send out error reports to us. This reporting capability execution can be controlled using this option.

 

•Automatically close alerts

 

  Cacadu alerts display critical information to the users. But using this option the users are able to suppress the generation of Cacadu alerts.

 

•Automatically start triggers at startup

 

  This option allows users to control the execution of triggers from the previous session.

 

•On Closing

 

  It is possible to have Cacadu homed in your system tray using "Send to System Tray" option.

 

•Default position for Cacadu's alert

 

  The position of the alerts generated in the Cacadu application can be controlled using this option
