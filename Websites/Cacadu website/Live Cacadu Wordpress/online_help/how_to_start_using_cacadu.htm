How to start using Cacadu

In this quick demonstration, we create a new task to run the calculator only one time at your desired time. You will simply understand what is a task, an action, and a trigger, and how to combine them to schedule your important tasks.

 

 

1.Click "Add Task" button to add a new task.

 

2.Write task name and click OK.

 

3.Double click task name to open its task flow...

 

4.Now, drag Start Application action from Actions Gallery to the selected Task Flow.

 

5.A new Start Application action window will be open. Write "calc" in File Name text box.

 

6.Now, Run Calc Task Flow has the start application action.

 

 

7.Click Triggers button to attach a trigger to the task.

 

8.Click Add Schedule Trigger button open the next dialog...

 

9.This is the last step. Set the start date for the trigger and click OK, and your task will run once at your specified start date.
