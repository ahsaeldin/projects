Advanced Triggers

The advanced triggers in Cacadu provides more advanced patterns that will allow a task to be performed at a specific date and time.

 

They are designed to give an administrator the flexibility to repeat the task at any required interval.

 

Each of the drop-down boxes allows the selection of one or more options. Selecting specific days, months or years will cause the task to execute every time that interval has passed.

 

The exception is the ordinal day of the week that will run the tasks when the specified day of the week appears the first, second, third or fourth time in a month.

 

An advanced trigger can be set to exclude holidays so that tasks are not executed when an office is closed.

 

Selecting multiple triggers such as a day and several months will provide better control over when the tasks are executed although only one advanced trigger is required.

 

Setting the start and end times will stop triggers from activating tasks outside of the date range provided.

 

 

 

 

 

Here is a complete list that describe all functions of every element in this form:

 

•Start Date & Time: Set the start date for the trigger to be fired.

 

•End Date & Time: Set the end date for the trigger to be fired.

 

•Repeat Time: Selectable combo box where you can select the required time(s) to fire the trigger.

 

•Repeat Day Of Month: Selectable combo box where you can select the required Day of month(s).

 

•Repeat Month: Selectable combo box where you can select the required month(s).

 

•Repeat Day Of Week: Combo box where user can select the required day of week(s).

 

•Ordinal Day Of Week: Very amazing feature, 1st *day name* of the month.. like "1st sunday of the month"

 

•Repeat year(s): year(s) of the execution.

 

•Exclude holidays: Activate Holidays tab, where user can set a holiday pattern for the trigger to not fire on.

 

Note: For "Repeat Day Of Month", "Repeat Day Of Week" and "Ordinal Day Of Week" check boxes; you can check only one of them at time.
