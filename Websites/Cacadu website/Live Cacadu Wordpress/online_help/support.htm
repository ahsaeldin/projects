Support

The Cacadu community features a support group to help you solve any difficulties you might having with learning to use Cacadu. And the support forum can help you use it as well as possible. Cacadu documentation is also available to help.

 

If you would like to report issues to the Cacadu team, use the online chat system to talk with a representative of the Cacadu team. A support ticket system is also available to help you resolve any difficulties you encounter, and the Cacadu team is quick to respond.

 

Cacadu has pages on Facebook, Twitter and Google+, and the Cacadu blog is available to keep you up-to-date with the latest Cacadu developments.

 

Finally, the video tutorials are a great place to start learning basic of Cacadu and some advanced features.
