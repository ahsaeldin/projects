Overview

 

Cacadu is an automation application that enables you to schedule Windows. In Cacadu, users can define tasks, and each task can contain one or more actions. Each action is a windows event that users are able to configure in Cacadu.

 

Cacadu provides extreme control over the task configuration. It is possible to define Task variables to configure each task and the global variables and constants are used to control all the tasks. It is possible to check task details using the Task properties window, and also modify the properties through the properties window. In Cacadu it is possible to pause and resume tasks, and this feature is an indispensable tool for configuring the tasks.

 

The advanced features in Cacadu allow controlling sequencing of actions in a task, pausing between two actions in a task and delay an action in the task. It goes further, allowing error handling for the failures during task execution. It is possible to configure fallback actions for the task in the event of task failure. This is a great feature indeed where users can perform clean up actions as a response to task failures.

 

The tasks scheduling is controlled by Triggers in Cacadu. The triggers are schedulers which allow users execute the tasks and actions at desired time. Trigger calendar in Cacadu serves this purpose, and it is extremely convenient to use.

 

Cacadu is very detailed in capturing history of the application usage. For each action performed, Cacadu maintains a detailed history log to be for future reference. The advanced control option such as one click application control is also very impressive.

 

Summing up, Cacadu is an extremely powerful windows task scheduler, but at the same time is extremely convenient to use.
