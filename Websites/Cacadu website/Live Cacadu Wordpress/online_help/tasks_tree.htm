Tasks Tree

The core of the Cacadu task management is the Tasks Tree. The Tasks Tree allows quick access to the central features

of the Cacadu application.

 

Task Tree Actions:-

 

A user who clicks on a task node from the My Task pane will have a pop-up menu appear with access to the following actions:

 

oSave

 

oTriggers

 

oTask Flow

 

oTask Properties

 

oTask List

 

oRun

 

oStop

 

oAdd Task

 

oRename Task

 

oDelete Task

 

oAdd Group

 

oExport Task

 

oImport Task

 

A user who clicks on a group node will have access to the following actions:-

 

oTask List

 

oAdd Group

 

oRename Group

 

oDelete Group

 

oExport Tasks

 

oImport Tasks

 

Additionally, the search box in the My Tasks pane allows location of a task by entering part of the name or by clicking on its icon. Lastly, as with all window panes in Cacadu, the Task Tree's size and screen position are customizable; it can be moved or docked in a corner or simply allowed to float.
