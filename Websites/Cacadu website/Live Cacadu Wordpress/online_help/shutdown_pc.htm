Navigation:  Actions > System >

Shutdown PC

This action enables you to shutdown your PC along with other shutdown options. You can select to Log Off, Hibernate, sleep and restart your PC.

 

 

The advanced option in this action enables you to set additional parameter...

 

 

•Is forced: Check to force the action.

 

•Confirm before execute: Asks you just before execute.

 

•Allow Cancel: Allowing you cancel the action.

 

•Set a period before execution: Choose a delay period in seconds before start executing the action.

 

•You can right any comments about the action in the comments text box. The comments will be displayed beside action's name in the Task Flow.

 

Note: Enabled check box enables you to disable/enable the action.
