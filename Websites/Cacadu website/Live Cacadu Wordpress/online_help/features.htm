Features

Cacadu includes different powerful features. The most important features are:

 

•Easy and straightforward UI.

 

•Creating tasks. Tasks that can be considered as groups of actions. They are essentially a bundle of actions that can easily be scheduled to be executed at a certain time.

 

•Predefined actions.

 

•Task Flow that lets you build and manage your tasks.

 

•Scheduled triggers that makes running tedious tasks very simple.

 

•Task's variables can be defined for each task configured in the application, and they can be modified during task execution.

 

•Global constants are visible to all the tasks defined in the application. They control the common behavior of the tasks.

 

•Global variables do the same job as the variables but can be associated with any or all the tasks.

 

•Importing and Exporting Tasks. It is possible to export all the tasks from one Cacadu application and import it into another.

 

•Start up Trigger Options. Cacadu provides the capability to control the automatic resuming of all the triggers at start up.

 

•Defining Alert Location. Alerts can be configured to be displayed on Top Left, Top Right, Bottom Left or Bottom Right.

 

•Task Properties Window. Cacadu provides Task Properties window which can be invoked for a task to modify its properties.

 

•Trigger’s calendar can be used to schedule when and when not to fire a particular task in the application.

 

•Fallback Actions On Failure. It is possible to define fallback actions for a task that would be executed if the task fails to trigger.

 

•Advanced Execution Control Of Tasks. Tasks can be paused during the execution phase and resumed at the desired time.

 

•Configurable Time. If needed by the users, it is possible to control the delay time between execution consecutive actions.

 

•Configurable Delay Time. In addition to the delay time between actions , it is possible to define delay time between actions.

 

•Aborting On Error. Task can be configured to abort its execution if one of the actions associated with the task fails.

 

•In Depth History Logging. History is maintained for all activities that can be performed through using Cacadu for references.

 

•Auto update Capability. The application is designed to auto update to the latest version released by Conderella.

 

•Total Application Control. It is possible to enable and disable the whole application with a single click.

 

•Configurable Response for Errors. Cacadu allows configuration of response actions to be executed when particular error is received during task execution.
