Change Cacadu Theme

Cacadu provides personalization of the application according to an individual's display comfort using its vast impressive collection of inbuilt themes and skins.

 

The most basic collections of skips can be directly reached from the status bar in Cacadu. Besides the basic collection , Cacadu allows skin selection from two special skin collection sets called Bonus and Theme Skins, which are also accessible from status bar. The skin selection can be reverted to default skin "Seven" anytime by selecting the "Default" option in the main menu. Also, Cacadu provides "Random" option that changes the skin automatically from the vast collection of available skins.
