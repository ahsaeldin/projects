ASTC 1.50 Features  

ASTC is four ActiveX controls in one Control

1. Tray Icon ActiveX Control Features:

  • Display/Hide your application icon in system Tray area.
  • Creates a custom animated icon in your taskbar.
  • Assign a tooltip for the icon.
  • Check the current state of the icon [visible or invisible].
  • Display a balloon tooltip in system tray even displaying on win9x.
  • Change the icon and tooltip in run time.
  • Ability to assign a popup menu for a particular event happens to the icon.
  • Track Popup menu so whenever your program lost focus the icon popup menu will be closed.
  • Show more than one icon in the system tray.
  • Supports custom events for the icon [LeftMouseUp, RightMouseUp etc...]

2. Tray List ActiveX Control Features:

  • Get all the information about the system tray icons [Application path, uID, Hwnd, hicon, tooltip, ucallbackmessage etc...];  
  • Hide any icon from system tray area.

3. SubClass ActiveX Control Features:

  • Intercepting Windows messages that your program normally wouldn't receive, so it extends your ability to process more windows messages and add new features to your application.

4. Balloon ActiveX Control Features:

  • Assign a balloon tooltip to a particular control or a window.
  • Customize the balloon BackColor and ForeColor.
  • Customize the Balloon Style [Balloon or Rectangle].
  • Custom events associated with the balloon [BalloonHide, BalloonLeftClick, BalloonRightClick and BalloonShow].
