What is the system tray or systray?
The system tray is a section located at one end of the taskbar in Windows 95, 98, Millennium, 2000, and XP. It is used to display the clock as well as the icons for certain programs.

What do you mean by STI?
STI is stand for System Tray I cons.
What is the STI 1.06 Type?
ActiveX control.
What is the Compatibility of STI 1.06 with .Net?
100% .NET Compatible.
What are the languages included in the sample code?
VB 6.0, VB.NET, VC#.NET, VJ#.NET

Overview of STI v1.06

STI is an ActiveX control which enables the developer to get all the information about system tray icons from his/her application, also STI supports hides any icon from system tray area.

             

What are the Benefits of using STI?

  • STI ActiveX control save your development time because if you want to know anything about system tray icons from your application, you will code a dozen of ugly win32 API function which will take you off from the main idea of your application.
  • Manage System tray icons.
  • Hide any system tray icon.
  • The STI 1.06 installer package contains samples code of using STI in the following languages Vb6.0, VC++6.0, VB.Net, VC#.NET and VJ#.NET.
  • FREE version-upgrades, bug-fixes and online support for registered users.

What is the Possible Uses of STI?

  • If you need to know the number of the icons displayed in system tray.
  • If you need to know the path of the application which adds a specified icon to the system tray.
  • If you need to get the handle /[hwnd] of the window which adds a specified icon to system tray.
  • If you need to hide an icon from system tray area.
  • If you want to insure that is a specified icon is displayed in systray while your application is running.
  • If you want to insure that is a specified icon is not displayed in systray while your application is running.
  • If you need to get the handle of an icon displayed in system tray.
  • If you need to get the tooltip associated with an icon displayed in system tray. 
  • and more ...
