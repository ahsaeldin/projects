Easy Window & System Tray Icons Hider will run on any computer systems that meet the minimum requirements specified here.

 

Minimum system requirements:

 

Processor Pentium II class at least 200 MHz

OS Windows 95, 98, ME, NT 4.0, 2000, XP, Vista

RAM 64 MB

Disk Space 3-5 MB of initial free disk space
