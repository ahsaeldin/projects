Hide Windows and System tray icons

 Top  Previous  Next

 

 

To Hide any window, use one of the following ways:

 

•Simply Click the Window Caption.
•Check the checkbox beside the window caption then click Hide button.
•You can use a shortcut key to hide the foreground window, the predefined shortcut key to hide the foreground window is Ctrl+Alt+H, Note that you can customize all shortcut keys as you wish.

 

 

 

You can hide any or all system tray icons by selecting the icon then pressing Hide command button, also you can hide all system tray icons using a shortcut key.
