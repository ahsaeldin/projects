How can I hide a window?
Check the checkbox beside the window caption then click Hide button or You can use a shortcut key to hide the foreground window, the predefined shortcut key to hide the foreground window is Ctrl+Alt+H, Note that you can customize all shortcut keys as you wish or Simply Click the Window Caption in the list. What is system tray or systray?
The system tray is a section located at one end of the taskbar in Windows 95, 98, Millennium, 2000, XP and Vista. It is used to display the clock as well as the icons for certain programs. Back to top

How can I hide a system tray icon?
