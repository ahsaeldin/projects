•Starts when Windows starts : Check this check box to Start Handy Startup Monitor at windows startup .

 

•Allow Startup Monitor : This option allow you to disable/enable startup monitor. If you uncheck this option, you won't  get notified if a new program added or deleted from windows startup. The check box is checked by default and if a new program added or deleted from system startup, a popup balloon will be displayed in the system tray to alert you about the change .

 

•Newly added applications to startup : Alert you if a new program added to windows startup . If you uncheck this box, you won't get notified if a new program added to windows startup .

 

•Removed applications from startup : Alert you if a program removed from system startup . If you uncheck this box, you won't get notified if a new program removed from windows startup .

 

Show Handy Startup Monitor in System Tray :

 

•Always : always display Handy Startup Monitor's icon in the system tray .

 

•When minimized : only display the icon in the system tray when the Handy Startup Monitor get minimized .

 

•Never : never display Handy Startup Monitor in the system tray .

 

•Minimize to system tray instead of closing : check this box to minimize Handy Startup Monitor to system tray when you minimize it instead of closing .
