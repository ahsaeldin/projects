What is the macro?
A sequence of keyboard strokes and mouse actions that can be recorded so that their playback can be activated by a single keystroke, keystroke combination, or mouse click. Back to top

What are the system requirements?
Intel Pentium, Celeron, AMD or compatible processor Windows NT4/2000/XP/2003/Vista/Windows 7 and .NET FrameWork 3.50 Back to top

How can I stop recording?
Press Ctrl+W (You can customize Stop Recording hotkey). Back to top

Is there is a version of Perfect Macro Recorder for Mac or Linux?
No, currently Perfect Macro Recorder can only run under the windows.  Back to top

How can I stop play backing?
Press Ctrl+B or Ctrl+Alt+Del or Ctrl+Esc. Back to top

I'd like to use Perfect Macro Recorder under windows 95, is it possible?
No, Perfect Macro Recorder only run under Windows NT4 or above. Back to top

Why Sometimes the high playback speed causes some unexpected results?
Because there is no delay time between events in the high play back speed. Back to top

What is the executable macro?
You can convert any macro to an EXE file, so the macro can be run stand a lone without Perfect Macro Recorder. Back to top

How can I terminate play backing an Executable macro file?
Press Ctrl+Shift+F2 keys or Alt+Ctrl+Del keys. Back to top
