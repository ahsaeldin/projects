Overview of Perfect Macro Recorder

 

Perfect Macro Recorder is a macro recording utility. It is simply a tool enables you to record mouse movements, clicks and keyboard's keystrokes the save to a file and later you can resend those activities automatically to Windows in a process called "Playing a macro".

 

A Macro is a set of sequential tasks or commands which can be reused repeatedly in a way that simplify the life. "Macro Recording" is a process for recording tasks or commendssequentially for re-use later!

Full Features List :

 

Records keystrokes, mouse events, and clicks for later use!

Macro script editor to edit and modify your macros

Smart playback

Repeat playback a certain number of times

Convert your macro to an EXE-file

Variable playback speed

Pause/Resume Recording

Pause/Resume Playbacking

Auto run on Windows startup

Save and load macros to/from Perfect Macro Recorder

Install/uninstall support
