Saving and loading a macro

Perfect Macro Recorder gives you the ability to save and load your macros

 

To save macro to a file :

 

- Click the floppy disk icon.

- Click on the "Exe" icon to save macro as a standalone EXE file.

 

 

To load a macro file :

 

- Open the macro file using the folder icon.

- Drag and drop the macro file into macro script list.

- Drag and drop the executable macro file into macro script list.

 

Tip: by Perfect Macro Recorder's settings, you can choose a default file name and path for

a default folder to automatically save macros files to it .
